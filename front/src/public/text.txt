/*Guia de Estilo: Dashboard Ambiental (Material UI)

Esta paleta foi desenhada para criar uma interface moderna, limpa e funcional, utilizando os princípios do Material Design.

1. Paleta de Cores (Tokens)

Cores Primárias e de Marca

Primary (Tecnologia/Confiança): #1A73E8 (Blue 600) - Usada para cabeçalhos e elementos principais.

Secondary (Limpeza/Ar): #00B8D9 (Cyan 600) - Ideal para detalhes e acentos.

Background (Fundo): #F8FAFC - Um cinza azulado muito leve que reduz o cansaço visual.

Paper (Cards): #FFFFFF - Branco puro para destacar os cartões do fundo.

Cores de Estado (Semânticas)

Estas cores devem ser usadas nos Chips, Medidores e Alertas:

Success (Seguro): #10B981 (Emerald 500)

Warning (Atenção): #F59E0B (Amber 500)

Error (Crítico): #EF4444 (Red 500)

Info (Dados): #6366F1 (Indigo 500)

2. Configuração do Tema (MUI)

Podes copiar este objeto para a configuração do teu createTheme:

const theme = createTheme({
  palette: {
    primary: {
      main: '#1A73E8',
      light: '#E8F0FE',
      dark: '#1659B1',
    },
    secondary: {
      main: '#00B8D9',
    },
    background: {
      default: '#F8FAFC',
      paper: '#FFFFFF',
    },
    text: {
      primary: '#1E293B',   // Slate 800 (Mais suave que preto puro)
      secondary: '#64748B', // Slate 500
    },
    divider: '#E2E8F0',
  },
  shape: {
    borderRadius: 16, // Bordas arredondadas modernas como nos teus cards
  },
  typography: {
    fontFamily: '"Inter", "Roboto", "Helvetica", "Arial", sans-serif',
    h4: {
      fontWeight: 800,
      letterSpacing: '-0.02em',
    },
  },
});


3. Guia de Aplicação Visual

Gráficos (ECharts)

Para que os gráficos combinem com o Material UI, utiliza estas cores para as linhas:

Temperatura: #FF5252 (Vibrante)

Humidade: #448AFF (Azul Água)

Qualidade do Ar: #00E676 (Verde Neon)

Pressão: #7C4DFF (Roxo Profundo)

Tipografia e Contraste

Títulos: Usa color: "text.primary" para máxima legibilidade.

Subtítulos/Labels: Usa color: "text.secondary" para criar hierarquia.

Bordas: Mantém as bordas dos cards finas (1px solid) com a cor divider em vez de sombras pesadas para um visual "flat" e moderno.*/
